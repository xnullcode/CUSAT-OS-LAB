# Loadable Kernel Module

Implement and add a loadable kernel module to the Linux Kernel. Demonstrate using insmod, lsmod, and rmmod commands.

## Topics
- Writing a basic kernel module
- Compiling a kernel module
- Loading a module using insmod
- Listing modules using lsmod
- Removing a module using rmmod
